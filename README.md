# Childcare Tracker

A single-page web app to track babysitter hours and daycare attendance.

## Features

- **Babysitter tab**: Log clock-in/clock-out sessions with hourly rate. Supports open sessions (save with just clock-in, close later).
- **Daycare tab**: Log attendance days with drop-off / pick-up times, supply refill requests (diapers, wipes, etc.), and incident flags. Fixed $200/week rate.
- **Dashboards** on both tabs with date-range filtering (This Week / This Month / All Time).
- **Dark mode** UI.
- All data stored in browser `localStorage` — no server required.

## Usage

Open `index.html` in a browser, or use the hosted version on GitHub Pages.

## Tech

Pure HTML / CSS / vanilla JavaScript. No build step, no dependencies.
